# Project Context

This project is a Personal Task Web App built with:

- Next.js App Router
- Supabase
- Tailwind CSS

Main features:

- authentication
- workspace management
- board management
- task CRUD
- member management
- role-based permissions
- dark mode

Roles:

- owner
- admin
- member
- viewer

UI direction:

- modern SaaS
- clean and minimal
- practical over flashy
- easy to demo
- suitable for a student portfolio but should feel production-ready

Important implementation preference:

- keep code changes focused
- avoid large refactors unless necessary
- keep the dashboard structure understandable
- prioritize clarity and maintainability
